# Lannister Pay

### Getting Started

1. ##### Install Redis with redisSearch module
    `sudo docker run -p 6379:6379 redislabs/redisearch:latest `

2. ##### Add .env file with content below
    `
    PORT=8000
    `
    `
    NODE_ENV=development
 `

 3. ##### Install dependencies
    `npm install`
4. ##### Start App
    `npm start`
